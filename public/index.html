Comparing Images with Standard File Format Compression

Objective: To identify a baselevel of compression that we can apply to all images created for websites that facilitates the fastest loading possible without comprimising the visual or artiistic quality of the images. Pain point; despite there being great new lossless compressable image formats avaiable to use on the web aimed at increasing performance their depolyemnt is tricky and difficult due to the fact that many older browsers, which need to be supported, are not capable of handling these formats.

Notice that the compressions percentages are not indicative of the file size reduction percentages
